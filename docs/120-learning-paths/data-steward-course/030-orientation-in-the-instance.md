This module teaches a steward how to navigate the instance and build a repeatable starting point for daily work.

## Learning objective

Use business domains, search entry points, saved searches, and result views to move confidently around the instance.

## What to learn

- how the search box behaves;
- how business domain scoping changes results;
- how tabular and tile views support different stewardship needs;
- how saved searches reduce repeated setup work.

![Search box]({{ "/assets/images/key-terms-and-features/search-box.png" | relative_url }})

![Search results page]({{ "/assets/images/key-terms-and-features/search-results-page-default.png" | relative_url }})

## Exercises

1. Open the search box and switch between recent searches and saved searches.
1. Select a business domain and re-run the same keyword search.
1. Switch between tile and tabular views.
1. Add one useful column to tabular results.
1. Save one search that would be useful for future stewardship work.

## What good looks like

The learner can explain where to start when a stakeholder says, “something looks wrong in this domain.”
