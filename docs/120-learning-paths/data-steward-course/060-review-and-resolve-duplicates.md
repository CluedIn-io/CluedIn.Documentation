This module teaches a steward how to review duplicate candidates and make a defensible decision.

## Learning objective

Understand the deduplication flow well enough to review evidence, compare conflicting values, and explain the final choice.

## What to learn

- deduplication project creation;
- matching rule intent;
- group review;
- conflict resolution and merge preview.

## Exercises

1. Open an existing deduplication project.
1. Review one matching rule and explain what it is actually checking.
1. Open one duplicate group and compare conflicting values.
1. Document which values should survive and why.
1. Review the merge preview before approval.

## Steward judgment to reinforce

Deduplication is not only a feature workflow. It is a record-level decision process. The steward should always be able to explain the business rationale behind the selected values.
